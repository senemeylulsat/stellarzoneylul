# NFT Ticket Collection Platform

An innovative platform that turns event tickets into NFTs, allowing users to collect, showcase, and trade them in a digital profile. Perfect for concerts, sports matches, university events, museums, and exhibitions.

## Features

- **NFT Tickets:** Event tickets are minted as NFTs for digital collection.  
- **Profile Collections:** Users can store their attended event tickets in their personal profile.  
- **Custom Badges:**  
  - **Football Clubs:** Match attendance badges  
  - **Universities:** Event commemorative NFTs  
  - **Museums / Exhibitions:** Digital entry tickets
